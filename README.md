
# Password Generator

This is an app that can generate a password for you.
You can choose if you want to have numbers and symbols in your password.
The password can also get saved in a passwords.txt file.


## Usage/Examples

Download the files
# For windows
```command
./main-win.exe -h
```

# For Mac OS
```command
./main-macos -h
```

# For Linux
```command
./main-linux -h
```
